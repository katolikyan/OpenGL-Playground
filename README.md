# OpenGL Playground
This is just my sendbox to explore OpenGL.

## Usage
```bash
$> make
$> ./playground
```

`w` `a` `s` `d` - Move __camera__

`r` - Reset __camera__ position

`u` - Lock / unlock __camera__ 
> when camera is locked you are able to resize the window
